## Description
Budget Buddy is a simple web application built using HTML, CSS, and JavaScript. It allows users to track their expenses and manage their budgets effectively.

## Installation
1. Clone the repository: `git clone https://github.com/your-username/budget-buddy.git`
2. Open the `index.html` file in your web browser.

## Usage
1. Enter your total budget in the "Budget" section and click "Set Budget".
2. Enter the title and cost of each expense in the "Expenses" section and click "Check Amount".
3. View your total budget, total expenses, and remaining balance in the "Output" section.
4. Edit or delete expenses from the list as needed.

## Features
- Set a total budget
- Add and track expenses
- Edit or delete expenses
- View total budget, total expenses, and remaining balance
- Pie chart visualization of expenses
